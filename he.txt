ls(ll) : 查看当前目录的文件


cat he.txt  查看文件内容

tail -n 1 he.txt 查看文件内容末尾第一行

git status  查看文件状态      红色文件表示文件在工作区，没有被追踪到

 

git status  查看文件状态      绿色文件表示文件在暂存区，文件被追踪到 



git rm --cached he.txt 将文件从暂存区删除

将暂存区的文件提交到本地库
git commit -m "日志信息" 文件名
git commit -m "日志信息" he.txt

查看版本号信息   git reflog

git log 查看详细提交信息 


 版本穿梭  git reset --hard "版本号"


git branch -v   查看分支

git branch hot-fix  创建一个hot-fix分支

git checkout hot-fix 切换分支


合并分支  站在matser上   git merge hot-fix

git-hub HTTPS  https://github.com/lovettsh/gitprofile.git

创建远程库别名 

git remote -v  查看创建的远程库别名
git remote add gitprofile https://github.com/lovettsh/gitprofile.git   别名是gitprofile

推送远程库
git push gitprofile master    从本地库推送远程库后面需要跟  别名、分支

远程库拉取到本地库
git pull gitprofile master    从远程库拉取到本地库后面需要跟  别名、分支
